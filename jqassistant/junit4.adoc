==== Überprüfung des Klassennamens

[[junit4:TestClassName]]
.Alle JUnit4-Testklassennamen müssen mit "Test" enden.
[source,cypher,role=constraint,requiresConcepts="junit4:TestClass"]
----
MATCH
  (c:Junit4:Test:Class)
WHERE NOT
  c.name =~ ".*Test"
RETURN
  c as InvalidTestClass
----
