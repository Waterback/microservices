=== Build mit Maven
[source,bash]
.Wertpapierverwaltung bauen
----
$ mvn clean install
----

[source,bash]
.jQAssistant auslösen
----
$ mvn jqassistant:reset jqassistant:scan jqassistant:analyze jqassistant:report
$ mvn jqassistant:server
----
