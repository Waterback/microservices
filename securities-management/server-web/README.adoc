== Requirements
* Java SE 8
* Apache Maven, Version 3.3.3
* Eclipse Java EE IDE for Web Developers, Version Mars Release (v4.5.0)

== Verwendete Bibliotheken
* Spring Boot
** Spring Boot Actuator
*** Dropwizard Metrics
* Spring AOP / AspectJ
* Spring Data JPA
** Hibernate
** HSQLDB
* Spring MVC
** Thymeleaf
** Bootstrap / jQuery
